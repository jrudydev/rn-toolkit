# Assessment Practice

Skeleton project for the Link Management assessment. Use this as your starting point for the coding challenge.

## Getting Started

1. **View the assessment requirements** at [Challenge Hub](https://astacinco.com/hub)
2. Open this folder in your IDE
3. Run `npm install`
4. Run `npm start`
5. Build in `App.tsx`

## Files

| File | Purpose |
|------|---------|
| `App.tsx` | Starter code - your work goes here |
| `data/links.ts` | Mock link data |
| `i18n/en.ts` | Translation strings |

## Assessment Details

Full requirements, cheatsheets, and solutions are available in the **Challenge Hub** app:

- **Web**: https://astacinco.com/hub
- **Assessment**: Link Management Screen
- **Time**: 90 minutes
- **Difficulty**: Medium

## Packages Used

- `@astacinco/rn-primitives` - UI components
- `@astacinco/rn-theming` - Dark/light mode
- `@astacinco/rn-i18n` - Pluralization
